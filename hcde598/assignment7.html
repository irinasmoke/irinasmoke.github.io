Final Project Planning

Concept

For the final project, I want to make a set of planters for some succulents that I have had for a few months. I will make a set of 3 planters and 3 stands for those planters of different heights, but all with a similar design.

I was inspired by this set of planter stands by the designers at bernabeifreeman. I plan to create similar stands out of plywood, but on a smaller scale. I plan to make 3 stands of different heights, all with a similar pattern on the sides.

To go inside the stands, I will 3D print and make casts of this planter from thingiverse. I plan to make a silicone mold and cast them out of concrete. The file is a bit small (only 70mm in diameter), and I think I will scale it up a little bit.

In summary, the techniques I will use are:

  • 3D printing the planter
  • Molding & casting the planter
  • CAD modeling the stands, probably in OnShape
  • Laser cutting the stands

Contingency Plans

I have built in contingencies for things going wrong in the table above, such as leaving time to 3d print and laser cut multiple times if necessary. Other things I forsee may go wrong:

  • If for some reason the concrete does not work, I will use Plaster of Paris like I did for the previous assignment.
  • If for some reason the plywood doesn't work well, or I don't like how it looks, I will swich to acrylic. I have some leftover from a previous assignment, and I can buy more at the MILL if necessary.
